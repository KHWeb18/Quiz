## 23. 중복 방지 문제(Challenge)

```make

하나는 5 ~ 10의 범위를 가지고 다른 하나는 7 ~ 10의 범위를 가진다.
22번 예제를 응용하여 풀 수 있는 문제고 다소 난이도가 높은 문제다.
ex) 5, 6, 7, 8, 9, 10이 모두 출력되어야 하고 또한 7, 8, 9, 10이 출력되어야 한다.
    여기서 중복이 발생하면 안됨
```
